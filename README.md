# Fakespeare

This is a gem that generates silly, code-related Shakespeare quotes. Written as part of my RubyConf 2021 talk, `To mock, or not to mock?`

## Usage

```
Fakespeare.generate_quote
# => "Tempt not a desperate gem"
```
